Add/Edit Profile - 32/64-bit Debugger

Enter the path to cdb.exe for the appropriate version of the Debugging Tools for Windows. If you are running StackHash on a 32-bit computer then the 64-bit Debugger tab will be disabled.

Enter your default symbol and binary (executable) search paths. Each path must be separated by a semi-colon. Symbol servers are supported. We recommend including both the Microsoft public symbol server and the internal symbol store for each product that you will be analyzing with StackHash.

Note that these search paths can be supplemented or replaced entirely in StackHash scripts. See Scripting for more information.
