Plugin Diagnostics

All Plugins

Lists all available plugins, including plugins that failed to load. The Last Exception column provides more information for load failures.

Select a plugin in the list to populate the diagnostics sections below. Diagnostics are provided by the plugin. Consult the help or instructions provided with the plugin for assistance with interpreting the diagnostic information.

Global Diagnostics

Reports diagnostics from the selected plugin that apply to all profiles.

Profile Diagnostics

Reports diagnostics from the selected plugin that apply to the current profile. If this section is disabled then either the current profile is inactive or Plugin Diagnostics was opened from the Service Status window without first selecting a profile.
