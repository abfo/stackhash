Add/Edit Profile - Database

Connection Timeout: Number of seconds to wait for a connection to the database to open. Value must be between 5 and 120 seconds.

Events Per Block: Number of events to retrieve from the database in one call. Value must be between 10 and 10,000. Changing this setting from the default of 100 is not recommended unless advised by StackHash support.

Minimum Pool Size: Minimum number of reserved connections to maintain to the database. Value must be between 2 and 100. Set this to at least one plus the number of StackHash clients you expect to be accessing the service simultaneously.

Maximum Pool Size: Maximum number of reserved connections to maintain to the database. Value must be between 2 and 100 and must be at least as large as the minimum pool size.
