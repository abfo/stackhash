Client Options - Service Connection

If you installed the StackHash client and service on the same computer the default host and port do not need to be changed. If the service is running on a different computer you must enter the host (network name or IP address) and port to connect with. If you are not sure what to enter check with the person responsible for administering your StackHash server.

The list of Profiles may be empty if you are not connected to the StackHash service or there are no active profiles available. Click the Service Options button to fix this (Note: The service can only be configured by a client running on the same computer as the service.)

Click the Service Credentials link to specify the account to use when connecting to the StackHash Service. By default your current Windows credentials are used.

If you change the service host and/or port the profile list is cleared. Click Refresh to update the list with available active profiles.

StackHash can write a troubleshooting log to your desktop directory. In addition to the client log, on the Service Connection tab you can also enable logging for the service.
