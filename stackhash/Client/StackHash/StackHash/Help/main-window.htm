Main Window

The main window displays four different views as you drill down into the mapped products:

  • Mapped Products
  • Event List
  • Event Details
  • Cab Details

Menus

File

Export Export the mapped product list; events for the currently selected product; or events, hits (event infos) and cab information for the currently selected product.
Upload Mapping File   Uploads a new product mapping file to WinQual.
Exit   Closes the StackHash client. The service will continue to run and will synchronize as scheduled.

View

Back Returns to the previous view in the main window.
Home Returns to the Mapped Products view in the main window.
Show Disabled Products   Toggles display of products that have been disabled for synchronization. To change the synchronization state right-click the product in the Mapped Products view.
Show Events Without Cabs   Toggles display of events without associated cab files.
Refresh   Refreshes the current main window view.

Debug

Run Script Immediately run a script on the currently selected cab.
Script Manager Add, edit, delete and test StackHash debugger scripts. See Script Manager.
Start Debugging Launches the currently selected cab in the default debugger. Set the default debugger in Tools -> Client Options -> Debuggers.
Debug Using   Launches the currently selected cab in the debugger of your choice.
Extract Cab Extracts the contents of the currently selected cab to a folder of your choice.
Download Cab   Downloads the currently selected cab.
Open Cab Folder   Open the folder containing the currently selected cab file (this menu item is only available if the StackHash Client is running on the same computer as the StackHash Service).

Tools

Synchronize Now Connects to WinQual and adds new products, events, hits (event infos) and cabs. StackHash analyzes new cabs once the sync completes.
Resynchronize Now Connects to WinQual and updates all products, events, hits (event infos) and cabs in addition to adding any new information. StackHash analyzes new cabs once the sync completes.
Cancel Synchronization   Stops the current sync as quickly as possible.
Synchronization Report   Shows the result of the most recent WinQual synchronization. See Synchronization Report.
Synchronize with Plugins   Sends all WinQual data to one or more plugins. See Select Plugins.
Search Builder Allows you to search and filter events. See Search Builder.
Service Status Shows the current state of service tasks for each configured profile, including statistics on task success and failure. See Service Status.
Service Options Configure the StackHash service. This option is only available if you are running the client on the same computer as the service. See Service Options.
Client Options Configure the StackHash client. See Client Options.

Status Bar

The left hand side of the status bar shows client activity. The status will be Ready when you can use the client. The right hand side shows service activity. During synchronization the service status will let you know when events and cabs have been downloaded for each product. You will need to refresh the client view to see any new data (hit Ctrl-R, or View -> Refresh).

Personalization

The panes for each view can be resized. Hover your mouse over the space between each pane until the cursor changes to a left/right or up/down arrow and then click and drag to resize.

The columns in all lists can be resized and reordered. Click the column header and drag to change the order of the columns in each list.

StackHash saves the main window position, pane sizes, list column sizes/order and toolbar positions between sessions.
