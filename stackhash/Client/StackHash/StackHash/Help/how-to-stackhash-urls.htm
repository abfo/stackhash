StackHash URLs

StackHash URLs can be used to navigate directly to a product, event or cab. A StackHash URL takes the following form:

stackhash://[Profile Id]/[Product Id]/[Event Id]:[Event Type]/[Cab Id]

For example:

stackhash://0/11111/2222222222:Special%20Exception/3333333333

To obtain a StackHash URL right-click a product, event or cab and select the Copy URL option from the context menu.

You can navigate to a StackHash URL by pasting it into the StackHash search box and hitting enter. Outside of StackHash you can open StackHash to the URL item from a hyperlink in a browser, Windows Explorer or the Windows Run box (Windows Key + R).

If you are creating a StackHash URL (for example when writing a plugin) note that any spaces in the Event Type must be escaped as %20.
