Main Window - Event Details

The event details view shows a list of the cabs and hits (event infos) associated with an event.

The summary at the top right displays information about the selected product and event. Below the summary are notes relating to the event. Click Add Note to enter a new note.

Double-click a cab or click Show Details to show the Cab Details view.

A context menu is available when you right-click a cab. From this menu you can run a debugger script, launch the Script Manager, start debugging the cab, extract the contents of the cab to a folder of your choice or copy the StackHash URL.

Cabs

ID The WinQual Id for the cab.
Downloaded Yes if StackHash has downloaded this cab. You can download a cab by right-clicking it and choosing Download Cab.
Type WinQual Event Type Name associated with the cab.
Size Size of the cab.
OS Version Operating system of the computer for the dump file contained in the cab.
Architecture Machine architecture of the computer for the dump file contained in the cab.
.NET CLR version for the dump file contained in the cab (if applicable).
Process Uptime Process uptime for the dump file contained in the cab.
System Update System uptime of the computer for the dump file contained in the cab.
Created Local date and time that the cab was created on WinQual.
Modified Local date and time that the cab was last modified on WinQual.
Purged Yes if the cab has been deleted by StackHash. StackHash purges old cab to help you comply with the WinQual terms and conditions. By default cabs are purged after 180 days. You can change purge settings in Service Options.

Event Infos (Hits)

Hit Date Local date and time for the hit summary. Each summary row rolls up hits for a given date, operating system and locale.
Hits Number of hits. A hit is an actual report of the event through Windows Error Reporting.
OS Operating System for the hit summary.
OS Version Operating System version for the hit summary.
Language Language for the hit summary.
Locale Locale for the hit summary.
LCID Locale Id for the hit summary.
Created Local date and time that the hit summary was created on WinQual.
Modified Local date and time that the hit summary was last modified on WinQual.

Note that columns can be re-ordered in the client and so might not be in the order listed above. You can re-order columns by clicking and dragging the header.
