Add/Edit Profile - Plugins

Lists plugins that have been added to the profile.

Plugins can be enabled and disabled using the checkboxes in the Enabled column.

Click Add to add a plugin to the profile.

Click Settings to configure the currently selected plugin.

Click Plugin Help to open a web page provided by the selected plugin. If this link is disabled then the plugin has not provided a help URL.

Click Remove to remove the currently selected plugin. Removing a plugin also discards any settings.

Click Diagnostics to list available plugins including plugins that failed to load and any diagnostic information reported directly from each plugin.
