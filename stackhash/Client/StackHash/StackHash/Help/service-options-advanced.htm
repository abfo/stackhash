Service Options - Advanced

The service can only be configured by a client running on the same computer as the service.

Client Options

Client Timeout: The number of minutes that a client may be unused before the connection to the StackHash Service can be claimed by another client (booted). This setting helps prevent a client that is not currently being used from preventing other clients from accessing the StackHash Service. See Licensing for more information.
