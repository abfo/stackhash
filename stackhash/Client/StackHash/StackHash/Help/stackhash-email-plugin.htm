StackHash Email Plugin

Sends email when events and products are added from WinQual.

SMTP Settings: Enter the host (or IP address), port, username and password for the SMTP server you would like to use. Note that you may have to update firewall settings to allow the StackHash Service (StackHashService.exe) to access the SMTP server.

Email To: Recipient(s) of the email. If sending to more than one address separate using commas.

Email From: From address of the email.

Manual Mode: In manual mode emails will not be sent when events are added during synchronization. To send an email in manual mode right-click the event and choose Send event to Plugin -> StackHash Email Plugin. Enter True to enable manual mode or False if you would like email to be sent automatically.

Report new products: Enter True to send an email when a new product mapping is added (if Manual Mode is set to False). Enter False to never send email for new products.

To add a display name or envelope to an email address enter it like this: “StackHash Automatic Email <stackhash@mycorp.com>”.
