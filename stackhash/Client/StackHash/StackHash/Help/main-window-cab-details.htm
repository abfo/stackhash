Main Window - Cab Details

The cab details view shows the results of all debugger script runs on the selected cab and a list of the contents of the cab file. The box below the list displays the results from the selected script or a preview of the selected cab file.

The summary at the top right displays information about the selected product, event and cab. Below the summary are notes relating to the cab. Click Add Note to enter a new note.

A context menu is available when you right-click a script result. From this menu you can run the script again, delete the script results, run a debugger script, launch the Script Manager, start debugging the selected cab, extract the contents of the selected cab to a folder of your choice or copy the StackHash URL.

Debugger Script Runs

Script Name Name of the debugger script.
Run Date Local date and time that the debugger script was executed.
Script Type Auto for a system (read only) script, User for a user defined script.

Cab File Contents

File Name Name of the file (and subfolder within the cab if relevant).
Size Size of the file.

Note that columns can be re-ordered in the client and so might not be in the order listed above. You can re-order columns by clicking and dragging the header.
