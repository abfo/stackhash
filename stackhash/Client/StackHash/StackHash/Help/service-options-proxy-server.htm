Service Options - Proxy Server

The service can only be configured by a client running on the same computer as the service.

This tab configures the proxy server that the StackHash service uses to access the Internet. There are separate proxy server settings for the client.

If your proxy server requires a username and password but no domain leave the Domain field empty.

If StackHash fails to access the Internet and you're not sure what settings to use ask your network administrator.

StackHash scripts will fail to download symbols from the Internet if your WinHttp proxy is not configured correctly. If you experience this problem see Cdb from a Windows Service.
