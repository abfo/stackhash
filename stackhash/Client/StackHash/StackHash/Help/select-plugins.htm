Select Plugins

StackHash automatically sends new products, events, hits, cab information, notes and script results to each enabled plugin.

After adding a new plugin you may wish to send all of the above data stored in a profile to one or more plugins. This will seed plugins with existing data. This option may also help if the plugin is missing data, or if you have changed certain settings for the plugin.

Check the box next to each plugin that you want to synchronize and click OK.
