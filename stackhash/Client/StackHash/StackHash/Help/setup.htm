Setup Wizard

The Setup Wizard guides you through essential configuration options the first time you run StackHash on a new computer. To further configure StackHash after the wizard has completed use Client Options and Service Options from the Tools menu.

See Getting Started for important information about setting up StackHash.

If you're running the StackHash Client on the same computer as the StackHash Service the Setup Wizard has the following pages:

  • Welcome to StackHash
  • Add WinQual Account
  • Select Products to Synchronize
  • Set Collection Policies
  • More Options

When running the StackHash Client to connect to the StackHash Service remotely the Setup Wizard has the following pages:

  • Welcome to StackHash
  • Select Profile
