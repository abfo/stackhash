Setup Wizard - Welcome to StackHash

See Getting Started for important information about setting up StackHash.

If you installed the StackHash client and service on the same computer the default host and port do not need to be changed. If the service is running on a different computer you must enter the host (network name or IP address) and port to connect with. If you're not sure what to enter check with the person responsible for administering your StackHash server.

If you use a proxy server to access the Internet click Proxy Settings on this page to configure it now.

Click the Service Credentials link to specify the account to use when connecting to the StackHash Service. By default your current Windows credentials are used.
