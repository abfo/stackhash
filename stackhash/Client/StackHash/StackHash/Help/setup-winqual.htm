Setup Wizard - Add WinQual Account

The profile name is used to identify this profile. We recommend using your company or product name. You cannot use a profile name that is already in use by another profile.

Enter your WinQual username and password. If logon fails when you click Next check that you have entered your credentials correctly and try again. Also try logging in at winqual.microsoft.com. If you need to configure a proxy server click Back and click the link on the Welcome page.

Click Select to configure a cab folder and database for the profile using the StackHash Database Configuration Tool.

Enter the path to the 32 and 64-bit versions of cdb.exe from the Debugging Tools for Windows. If you are running StackHash on a 32-bit computer then the 64-bit path will be disabled.
