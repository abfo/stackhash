Upgrade Profile

When upgrading to Beta 7 or later you will be prompted to upgrade your profile(s) when StackHash loads.

Prior to Beta 7 WinQual data was stored in XML files. From Beta 7 on StackHash stores this information in a database (SQL Server 2005 or 2008).

Downloaded cab files are still stored in a folder, however the directory structure has changed to improve performance when a large number of cab files are present.

Click each listed profile and then click the Upgrade... button. This will launch the Database Configuration Tool which allows you to create a database and choose a cab folder for the profile.

If a profile is listed that you no longer need you can choose to delete it instead of upgrading (select the profile and click the Delete button).

If you do not already have SQL Server installed the Database Configuration Tool can install SQL Server Express 2008 R2 with the recommended settings for StackHash.

Upgrading profiles is required to use Beta 7 or later. If you close the Upgrade Profile window before all profiles are upgraded StackHash will exit.
