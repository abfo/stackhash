Choose a plugin to add to a profile.

Plugin: Lists available plugins to add to the profile. If a plugin you expect to be listed is not present see Plugin Diagnostics.

Sets Plugin Reference: Plugins can optionally set the Plugin Reference for an event (this field is visible in a column on the event list). Only one plugin that sets the Plugin Reference can be active for each profile.

Description: Displayed if provided by the plugin.

Click More information about this plugin to open a web page provided by the plugin. If this link is disabled then the plugin has not provided a help URL.
