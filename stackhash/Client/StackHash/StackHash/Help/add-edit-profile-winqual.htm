Add/Edit Profile - WinQual

The profile name is used to identify this profile. We recommend using your company or product name. You cannot use a profile name that is already in use by another profile.

WinQual Credentials

Enter your WinQual username and password. Click the Test button to ensure that StackHash can access WinQual using your credentials. If the test fails check that you have entered your credentials correctly and try again. Also try logging in at winqual.microsoft.com.

Cab Folder and Database Settings

Click Move or Edit Database to run the StackHash Database Configuration Tool. This will allow you to move your database and/or the contents of the cab folder. Note that moving to a new database can take a long time as each record is validated during the move.

Cab Folder and Connection String show the current name and location of the profile cab folder and database. Click Test to check that the StackHash Service can connect to the database.

The purge setting is the age of cab files that should automatically be removed in days. The default value of 180 will delete cab files and any extracted crash dumps from the index folder after six months. You should set this to comply with the data retention rules in your current WinQual agreement with Microsoft.
