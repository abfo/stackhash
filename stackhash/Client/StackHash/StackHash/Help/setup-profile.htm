Setup Wizard - Select Profile

Choose a profile to work with from the drop down list.

If no active profiles are listed use Tools -> Service Options from the StackHash Client running on the same computer as the StackHash Service to activate or add a profile. On this computer click Finish and then use Tools -> Client Options to select the desired profile.
