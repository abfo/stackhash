Script Manager

See Scripting for help on writing StackHash Scripts.

The Script Manager lists available scripts.

Script Name Name of the debugger script.
System Indicates if the script is internal to StackHash. You can view a system script but not modify it.
Automatic Indicates if the script is run automatically on new cabs (and existing cabs that have not yet been analyzed by the script).
Run On The type of dump that the script will be run on automatically (all dumps, managed dumps or native dumps). Note that a script can be manually run on any dump file regardless of this setting.
Last Modified The local date and time that the script was last modified.
Created The local date and time that the script was created.

If a cab is currently selected you can click Test to run the currently selected script.

To view or edit a script double-click the script in the list or click the Edit button (the View button for a System script). Click Add to create a new script; remove a script by clicking Delete.

You can rename a script by editing it and entering a new name.
