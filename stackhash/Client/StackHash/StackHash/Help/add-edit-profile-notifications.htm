Add/Edit Profile - Notifications

Server: Enter the host name or IP address of your SMTP server.

Port: Enter the port used by your SMTP server.

Username: If your SMTP server requires authentication enter the username.

Password: If your SMTP server requires authentication enter the password.

From: Enter the email address that notification emails should be sent from.

To: Enter one or more email addresses to send notification emails to. Use a comma to separate multiple email addresses.

Notifications: Check each type of report to sent. Notification emails will be sent when each operation starts and completes, with the exception of plugin failures.
